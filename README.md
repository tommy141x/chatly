# Chatly

Chatly is an open-source social platform similar to Discord or Guilded, built using Expo, React Native, Tauri, and Appwrite.

## Tech Stack

- 🚀 Expo & Tauri
- ⚛️ React Native
- 🎨 Tailwind CSS (NativeWind)
- 🧰 Shadcn/UI (React Native Reusables)
- 🎙️ LiveKit.io
- 🕙 Appwrite
